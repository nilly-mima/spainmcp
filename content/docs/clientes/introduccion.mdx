SpainMCP proporciona a quienes desarrollan agentes de IA una puerta de entrada al ecosistema completo de servidores [Model Context Protocol](/docs/conceptos/que-es-mcp) (MCP). Ya sea que tu proyecto involucre chatbots, agentes automatizados o copilotos inteligentes, **SpainMCP Connect** te permite incorporar servidores MCP de forma sencilla mediante un servicio totalmente gestionado.

## ¿Qué ofrece SpainMCP Connect?

- **Sin configuración OAuth** — Olvídate de URIs de redirección, IDs de cliente o secretos: todo se gestiona por ti
- **Renovación automática de tokens** — Tus credenciales se mantienen activas sin que tengas que hacer nada
- **Almacenamiento seguro de credenciales** — Cifrado de solo escritura para API keys y tokens
- **Sin estado en tu lado** — Todo el ciclo de vida de las conexiones se administra de forma automática
