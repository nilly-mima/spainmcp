SpainMCP es el marketplace de código abierto mas grande para servidores [Model Context Protocol](/docs/conceptos/que-es-mcp) (MCP). Explora y utiliza servidores MCP que dan a los LLMs la capacidad de buscar en la web, consultar bases de datos y mucho mas.

## ¿Qué puedes hacer con SpainMCP?

SpainMCP te permite encontrar, utilizar y publicar servidores MCP. Entre los servidores mas populares de la plataforma se encuentran:

- **Exa** — Realiza busquedas web en tiempo real, consulta perfiles de LinkedIn, lleva a cabo investigaciones en profundidad y mas
- **Context7** — Accede a la documentacion mas actualizada de los principales SDKs y frameworks directamente desde Cursor o Claude Code
- **Browserbase** — Controla un navegador web remoto mediante Stagehand

## Documentacion accesible para LLMs

Toda la documentacion de SpainMCP esta disponible en formatos pensados para ser consumidos por LLMs:

### Servidor MCP

Puedes acceder a la documentacion de SpainMCP a traves de MCP en `https://spainmcp.com/docs/mcp`. Tambien es posible utilizar el servidor MCP de Context7 para buscar dentro de nuestra documentacion.

### Plantillas para agentes

Implementaciones de referencia que te ayudan a construir agentes con SpainMCP:

- **Plantilla básica** — Construye servidores MCP básicos
- **App ChatGPT** — Construye apps para ChatGPT

### Formato Markdown

Todas las paginas son accesibles en markdown simplemente anadiendo `.md` al final de la URL. Por ejemplo, esta pagina se convierte en `https://spainmcp.com/docs/index.md`. Ademas, puedes copiar cualquier pagina en formato markdown directamente desde la documentacion.
