Con el **SpainMCP Gateway** puedes poner tu servidor MCP al alcance de cualquier cliente. Al publicar en SpainMCP, tu servidor queda visible en el directorio y disponible para que los usuarios se conecten de forma inmediata.

## Ventajas de publicar en SpainMCP

- **Distribución** — Obtén una página dedicada para tu servidor donde los usuarios pueden descubrir, probar e instalar tu MCP desde cualquier cliente
- **Analíticas** — Registra llamadas a herramientas, monitoriza patrones de uso y entiende cómo interactúan los usuarios con tu servidor
- **Cumplimiento de Spec** — El Gateway gestiona automáticamente el cumplimiento del protocolo MCP, el enriquecimiento de metadata y el caché
- **OAuth UI** — Generación automática de modales de autenticación para servidores que requieren configuración de usuario o API keys

## ¿Tu servidor ya esta en produccion?

Si tu servidor MCP ya se encuentra desplegado en otra plataforma, puedes registrarlo en SpainMCP usando el metodo URL. Basta con que exponga Streamable HTTP para ser compatible.
