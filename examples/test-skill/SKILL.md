# Test Skill — Generador de Saludos

Una skill sencilla de demostración que genera saludos personalizados.

## Cuándo activarse

- El usuario pide un saludo personalizado
- El usuario quiere dar la bienvenida a alguien
- El usuario menciona "saludo" o "bienvenida"

## Instrucciones

1. Pregunta el nombre de la persona a saludar
2. Pregunta el contexto (formal, informal, festivo)
3. Genera un saludo apropiado al contexto

## Ejemplos

**Informal:**
Input: Saluda a María de forma informal
Output: ¡Ey María! ¿Qué tal? Me alegro de verte por aquí 😄

**Formal:**
Input: Saludo formal para el Sr. García
Output: Estimado Sr. García, es un placer dirigirme a usted. Reciba un cordial saludo.

## Notas

Esta es una skill de prueba creada para verificar el flujo de publicación de SpainMCP.
